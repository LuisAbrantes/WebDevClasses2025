4. Conceitos Importantes

Child Elements

Elementos filhos são elementos HTML aninhados dentro de outros elementos (pais). A relação pai-filho é fundamental para navegação e manipulação do DOM.

PreventDefault()

Impede o comportamento padrão de eventos (como envio de formulários, cliques em links). Útil para validação personalizada e controle de fluxo.
